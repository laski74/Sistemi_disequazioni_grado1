Sistemi di Disequazioni Lineari

Es. 193 ★☆☆ Base
Risolvi il sistema: $$\begin{cases} -x + 8 \geq 9 \\ 3x > 12 + x \end{cases}$$
$$-x + 8 \geq 9 \implies -x \geq 1 \implies x \leq -1$$
$$3x > 12 + x \implies 2x > 12 \implies x > 6$$
Sistema IMPOSSIBILE: $$\emptyset$$
Es. 194 ★☆☆ Base
Risolvi il sistema: $$\begin{cases} 2(x-1) < 6 + x \\ 4x \leq -8 \end{cases}$$
$$2x - 2 < 6 + x \implies x < 8$$
$$4x \leq -8 \implies x \leq -2$$
$$x \leq -2 \text{ oppure } (-\infty, -2]$$
Es. 195 ★☆☆ Base
Risolvi il sistema: $$\begin{cases} 2r + 5 > 5r - 2 \\ 3(r+4) - r \leq 0 \end{cases}$$
$$7 > 3r \implies r < \frac{7}{3}$$
$$2r + 12 \leq 0 \implies r \leq -6$$
$$r \leq -6 \text{ oppure } (-\infty, -6]$$
Es. 196 ★☆☆ Base
Risolvi il sistema: $$\begin{cases} -2x + 3 \leq -9x \\ 5x - 1 < 1 \\ -4x + 12 > 3(4-x) \end{cases}$$
$$7x \leq -3 \implies x \leq -\frac{3}{7}$$
$$5x < 2 \implies x < \frac{2}{5}$$
$$-x > 0 \implies x < 0$$
$$x \leq -\frac{3}{7}$$
Es. 197 ★☆☆ Base
Risolvi il sistema: $$\begin{cases} 5x + 1 = 22x \\ x - 4x > 11 \\ x - 3 > -(1-x) \end{cases}$$
$$x = \frac{1}{17}$$
$$-3x > 11 \implies x < -\frac{11}{3}$$
$$-3 > -1$$ (FALSO!)
Sistema IMPOSSIBILE: $$\emptyset$$
Es. 198 ★★☆ Intermedio
Per quali valori di $$x$$ la funzione $$f(x) = -\frac{1}{2}x + 6$$ è non negativa?
$$-\frac{1}{2}x + 6 \geq 0 \implies x \leq 12$$
$$x \leq 12 \text{ oppure } (-\infty, 12]$$
Es. 199 ★★☆ Intermedio
Per quali valori di $$x$$ risulta: $$-2 < f(x) \leq 4$$ con $$f(x) = -\frac{1}{2}x + 6$$?
$$-\frac{1}{2}x + 6 > -2 \implies x < 16$$
$$-\frac{1}{2}x + 6 \leq 4 \implies x \geq 4$$
$$4 \leq x < 16 \text{ oppure } [4, 16)$$
Es. 200 ★★☆ Intermedio
Risolvi il sistema: $$\begin{cases} 8x > 0 \\ ax < 0 \text{ (con } a < 0\text{)} \end{cases}$$
$$8x > 0 \implies x > 0$$
$$ax < 0 \text{ con } a < 0 \implies x > 0$$
$$x > 0 \text{ oppure } (0, +\infty)$$
Es. 201 ★★☆ Intermedio
Risolvi il sistema: $$\begin{cases} 3x - 7x < 5 \\ -5x - 1 \geq -x - 5 \end{cases}$$
$$-4x < 5 \implies x > -\frac{5}{4}$$
$$-4x \geq -4 \implies x \leq 1$$
$$-\frac{5}{4} < x \leq 1$$
Es. 202 ★★★ Avanzato
Risolvi il sistema: $$\begin{cases} 5(x-2) - 3(1-2x) \geq 4 \\ 2x - 3 < \frac{5x - 2}{3} \end{cases}$$
$$11x \geq 17 \implies x \geq \frac{17}{11}$$
$$6x - 9 < 5x - 2 \implies x < 7$$
$$\frac{17}{11} \leq x < 7$$

Problemi Geometrici

Es. 203 ★★☆ Intermedio
Due numeri dispari consecutivi: la loro somma diminuita di 8 non supera 12, e il quadrato del maggiore meno il quadrato del minore supera 33.
Minore: $$2n+1$$, Maggiore: $$2n+3$$
$$4n - 4 \leq 12 \implies n \leq 4$$
$$8n + 8 > 33 \implies n > 3.125$$
I numeri sono: $$9$$ e $$11$$
Es. 204 ★★☆ Intermedio
Un rettangolo ha lati $$(x-2)$$ cm e $$(5x-1)$$ cm. Per quali valori il perimetro è compreso tra 42 cm e 126 cm?
$$P = 2[(x-2) + (5x-1)] = 12x - 6$$
$$42 \leq 12x - 6 \leq 126 \implies 4 \leq x \leq 11$$
$$[4, 11]$$
Es. 205 ★★☆ Intermedio
La base supera di 3 cm i $$\frac{1}{5}$$ dell'altezza. Per quali valori di $$h$$ il perimetro è compreso tra 120 cm e 300 cm?
$$P = 2\left[\frac{h}{5}+3 + h\right] = \frac{12h}{5} + 6$$
$$120 \leq \frac{12h}{5} + 6 \leq 300 \implies 47.5 \leq h \leq 122.5$$
$$[47.5, 122.5]$$
Es. 206 ★★☆ Intermedio
Un trapezio ha base maggiore tripla di quella minore e altezza pari a $$\frac{3}{4}$$ della base minore. Per quali valori l'area supera quella di un quadrato di lato $$b$$?
$$A_{\text{trap}} = \frac{3b^2}{2}$$, $$A_{\text{quad}} = b^2$$
$$\frac{3b^2}{2} > b^2 \implies b > 0$$
$$b > 0$$
Es. 207 ★★☆ Intermedio
Il lato di un quadrato è $$(x+3)$$ cm. Per quali valori il perimetro del quadrato con lato aumentato del 10% è minore di 22 cm?
$$P = 4.4(x+3) < 22 \implies x < 2$$
Vincolo: $$x > -3$$
$$-3 < x < 2$$
Es. 208 ★★☆ Intermedio
Tre numeri consecutivi: il doppio della loro somma diminuito di 5 non supera 35. Qual è il massimo della somma?
$$2(3n+3) - 5 \leq 35 \implies n \leq 5.67$$
Il numero più grande è $$7$$
Es. 209 ★★★ Avanzato
Un esagono regolare lato $$(3b+2)$$ cm. Per quali valori il perimetro supera il $$\frac{11}{9}$$ del perimetro di un quadrato di lato $$(6b-1)$$ cm?
$$18b + 12 > \frac{11}{9}(24b-4) \implies b < \frac{76}{51}$$
Vincolo: $$b > \frac{1}{6}$$
$$\frac{1}{6} < b < \frac{76}{51}$$
Es. 210 ★★★ Avanzato
Un triangolo ha lati $$x$$, $$x+1$$, $$x+2$$. Affinché esista il triangolo, per quali valori di $$x$$ è possibile costruirlo?
$$x + (x+1) > x+2 \implies x > 1$$
$$x > 1$$

Sistemi con Disequazioni Fratte

Es. 301 ★★☆ Intermedio
Risolvi: $$\begin{cases} \dfrac{x+6}{x-2} > 0 \\ 4x + 1 \leq 0 \end{cases}$$
Schema segni: $$x < -6$$ oppure $$x > 2$$
Lineare: $$x \leq -\frac{1}{4}$$
$$x < -6$$
Es. 302 ★★☆ Intermedio
Risolvi: $$\dfrac{x+3}{1-x} > 0$$
Schema segni: positiva per $$-3 < x < 1$$
$$(-3, 1)$$
Es. 303 ★★☆ Intermedio
Risolvi: $$\begin{cases} \dfrac{4-x}{x} \geq 1 \\ x + 2(x-1) < 4x + 5 \end{cases}$$
$$\frac{4-2x}{x} \geq 0$$ per $$0 < x \leq 2$$
$$x > -7$$
$$(0, 2]$$
Es. 304 ★★☆ Intermedio
Risolvi: $$\begin{cases} \dfrac{3x-1}{x+1} < 3 \\ x(x-2) < (x-1)^2 + x \end{cases}$$
$$\frac{-4}{x+1} < 0$$ per $$x > -1$$
$$x > -1$$
$$(-1, +\infty)$$
Es. 305 ★★☆ Intermedio
Risolvi: $$\begin{cases} \dfrac{1}{3}x - \dfrac{1}{2}(x-1) > -\dfrac{x+3}{6} \\ \dfrac{1}{3} \geq \dfrac{1}{x} \end{cases}$$
Prima: sempre vera
$$\frac{x-3}{3x} \geq 0$$ per $$x < 0$$ oppure $$x \geq 3$$
$$(-\infty, 0) \cup [3, +\infty)$$
Es. 306 ★★☆ Intermedio
Risolvi: $$\begin{cases} 2 - x \leq 0 \\ \dfrac{5}{2-x} \geq 0 \end{cases}$$
$$x \geq 2$$ e $$x < 2$$
$$\emptyset$$
Es. 307 ★★★ Avanzato
Risolvi: $$\begin{cases} \dfrac{3x-1}{x+1} \leq 3 \\ 4x \leq -(x+3) \end{cases}$$
$$x > -1$$ e $$x \leq -\frac{3}{5}$$
$$\emptyset$$
Es. 308 ★★★ Avanzato
Risolvi: $$\begin{cases} \dfrac{2x}{3x^2 + 4} \leq 0 \\ \dfrac{-4x}{x^2 + 1} \leq 0 \end{cases}$$
Prima: $$x \leq 0$$
Seconda: $$x \geq 0$$
$$x = 0$$
Es. 309 ★★★ Avanzato
Risolvi: $$\begin{cases} \dfrac{x^2 - 4}{x-1} \geq 0 \\ 2x - 3 < 5 \end{cases}$$
$$[-2, 1) \cup [2, +\infty)$$
$$x < 4$$
$$[-2, 1) \cup [2, 4)$$
Es. 310 ★★★ Avanzato
Risolvi: $$\begin{cases} \dfrac{x-3}{(x-1)(x+2)} > 0 \\ x + 1 > 0 \end{cases}$$
$$(-2, 1) \cup (3, +\infty)$$
$$x > -1$$
$$(-1, 1) \cup (3, +\infty)$$
